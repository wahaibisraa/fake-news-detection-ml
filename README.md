# 📰 Fake News Detection System

## 📌 Overview
This project is an NLP-based machine learning system designed to detect fake news using textual data. It applies text preprocessing, feature extraction, and classification techniques to identify misleading or false information.

---

## 🚀 Features
- Detect fake vs real news using machine learning
- Text preprocessing and cleaning pipeline
- TF-IDF feature extraction
- High classification accuracy (**92.46%**)
- Simple and efficient implementation

---

## 🛠️ Technologies Used
- Python  
- Scikit-learn  
- Pandas, NumPy  
- Natural Language Processing (NLP)  

---

## 📊 Model Details
- Feature Extraction: TF-IDF  
- Model: Random Forest Classifier  
- Accuracy: **92.46%**
